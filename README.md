## 支持的功能

- [x] 固定左/右列
- [x] 单元格合并
- [x] 表头可拖拽改变大小
- [x] 表格头部合并
- [x] 表格单元格编辑
- [x] 表格单列排序
- [x] 表格筛选
- [ ] 数据分组
- [ ] 可展开表格
- [ ] 树形表格
