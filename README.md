# 安装使用

```
npm install @lifhcp/rc-grid --save
```

```
yarn add @lifhcp/rc-grid
```

## 功能对比

|--------|-------- |-------  |---------        |------              |-----------   |
|固定左   | ✅     | ✅       | ✅              |✅                  | ✅
|右列     | ✅     | ✅       | ❌              |✅                  | ✅ 
|列可拖拽改变大小| ✅ | ✅      | ✅              |⚠️ 需要自定义列头     | ✅ 
|列头合并   | ✅     | ✅      | ❌              | ✅                 | ✅
|虚拟滚动   | ✅     | ✅      | ✅              |⚠️ 需要重写渲染       | ✅
|单元格编辑 | ✅     | ✅       | ✅              | ⚠️ 需要重写cell     | ✅
|可展开表格 | ✅     | ✅       | ✅              | ✅                 | ❌
|树形表格   | ✅     | ✅       | ✅              | ✅                 | ❌
|开源协议   | MIT    | MIT      | MIT             | MIT                | 商业
